Pinball

A month long prototype sprint to create a pinball game from scratch in Unity. This project includes core pinball features such as bumpers, catch and hold point payouts, drop targets, multipliers, and a black hole area to showcase a prototype sprint using Unitys 3D physics system.

The goal for this project was to create a complete gameplay loop with polish in as short amount of time as possible.

The key takeaway from this is learning how to manipulate Unitys 3D physics system in a way that will achieve the desired gameplay feel, while also keeping it in a controlable state for consistent and predictable outcomes.
